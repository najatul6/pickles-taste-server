# 📖 About

Pickles Taste API is a powerful backend service developed using the MERN stack. It provides secure RESTful APIs for managing products, authentication, users, orders, and application data while ensuring high performance and scalability.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/najatul6/pickles-taste-server.git
```

Move into the project

```bash
cd pickles-taste-server
```

Install dependencies

```bash
npm install
```

Run the server

```bash
npm run dev
```

or

```bash
npm start
```
